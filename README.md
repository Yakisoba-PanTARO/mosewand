# Mose's wand MOD

## はじめに
モーセの如く、海やマグマを割る杖を追加する MOD です。  
Minecraft の MOD 紹介のサイトを見ていたら、このような MOD があったので、勢いで作ってみました。  
使い道はよく分かりませんが、海底探索とかには使えるかもしれません。  

## レシピ
レシピは以下のように、`group`の`book`が有効なアイテムと、`default:stick`を並べます。
